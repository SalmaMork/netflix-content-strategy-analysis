# 📊 Netflix Content Strategy Analysis

## 📌 Project Overview

This project analyzes Netflix's content catalog to uncover trends in content types, growth over time, ratings distribution, and global presence.

The goal is to understand Netflix’s content strategy using data analysis techniques in Python.

---

## 📂 Dataset

* Source: Netflix Titles Dataset
* File: `netflix_titles.csv`

---

## 🛠️ Tools & Technologies

* Python
* Pandas
* Matplotlib
* Jupyter Notebook

---

## 🔍 Key Analysis Performed

* Data cleaning and preprocessing
* Movies vs TV Shows comparison
* Content growth over time
* Country distribution
* Genre analysis
* Ratings distribution

---

## 📈 Key Insights

* Netflix expanded rapidly after 2016
* Movies dominate the platform
* Strong focus on international content
* Content mainly targets mature audiences
